Installation

Bereitstellung der Windows-Version mit stabiler Ordnerstruktur und ohne zusätzlichen Build-Prozess.

Info: Dieser Abschnitt beschreibt bewusst nur die Bereitstellung der lauffähigen Anwendung. Entwicklungssetup, Datenbankeinrichtung und API-Konfiguration werden getrennt behandelt, damit Nutzung und technische Vorbereitung nicht vermischt werden.

Ziel dieses Abschnitts

Diese Seite beschreibt die Inbetriebnahme der bereitgestellten Windows-Version von liveXtrem. Der Schwerpunkt liegt auf einer reproduzierbaren Erstinstallation für Präsentationen, Abgaben und normale Nutzung. Dadurch lässt sich die Anwendung testen, ohne vorher eine Entwicklungsumgebung oder zusätzliche Python-Abhängigkeiten einrichten zu müssen.

Voraussetzungen

  • Windows 10 oder neuer
  • Internetverbindung
  • Berechtigung zum Entpacken von ZIP-Dateien und zum Erstellen einer Desktop-Verknüpfung
  • Ein dauerhaftes Zielverzeichnis, in dem der entpackte Anwendungsordner unverändert liegen bleibt

Da die Anwendung als vorkompilierte EXE bereitgestellt wird, müssen auf dem Zielsystem keine Python-Pakete manuell installiert werden. Entscheidend ist jedoch, dass die entpackte Ordnerstruktur vollständig erhalten bleibt. Die Datei livextrem.exe darf deshalb nicht aus ihrem Ordner herauskopiert werden, weil weitere Ressourcen relativ zu diesem Speicherort geladen werden.

Schritt-für-Schritt-Anleitung

1. Archiv herunterladen

Laden Sie die Datei livextrem_v1.0.zip aus dem Repository herunter. Das Archiv enthält die ausführbare Anwendung sowie die für den Start benötigten Zusatzdateien.

2. Archiv an einem festen Speicherort entpacken

Entpacken Sie die ZIP-Datei in ein Verzeichnis, das nicht versehentlich verschoben oder gelöscht wird. Geeignet ist ein fester Ordner auf dem lokalen Datenträger. Der Speicherort sollte anschließend nicht mehr verändert werden, weil die Anwendung auf Dateien innerhalb dieser Struktur zugreift.

Tipp: Suchen Sie die Datei livextrem.exe und erstellen Sie davon eine Desktop-Verknüpfung. So bleibt die Anwendung technisch an ihrem vorgesehenen Ort, während der tägliche Start komfortabel über die Verknüpfung erfolgen kann.
Wichtig: Löschen und verschieben Sie keinesfalls Dateien und Ordner von liveXtrem.

3. Anwendung starten

Starten Sie liveXtrem anschließend über die Verknüpfung. Wenn der Ordner vollständig entpackt wurde und keine Dateien verschoben wurden, lässt sich die Anwendung ohne weitere Installationsschritte öffnen.

Warum die exe im Ordner bleiben muss

Desktop-Anwendungen dieser Art werden häufig zusammen mit weiteren Ressourcen ausgeliefert, etwa Bildern, Theme-Dateien oder Konfigurationen. Wird nur die eigentliche Programmdatei verschoben, fehlen diese Abhängigkeiten am erwarteten Speicherort. Die Folge sind Startprobleme oder unvollständige Oberflächen. Die saubere Vorgehensweise besteht deshalb darin, stets den gesamten entpackten Ordner an seinem Platz zu belassen und lediglich eine Verknüpfung zu nutzen.

Abgrenzung zur Entwicklerinstallation

Die bereitgestellte ZIP-Datei dient der Nutzung der fertigen Anwendung. Informationen zur späteren Einrichtung von Datenbank, API-Zugängen und einem möglichen Quellcode-Setup werden bewusst getrennt auf der Seite Setup behandelt. So bleibt dieser Abschnitt auf die tatsächliche Inbetriebnahme der ausführbaren Version konzentriert.
