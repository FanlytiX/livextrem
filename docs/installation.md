# Installation

## Voraussetzungen
- Windows 10 oder neuer

## Download

[Download liveXtrem](https://github.com/FanlytiX/livextrem/blob/main/livextrem_v1.0.zip)

## Schritte

1. livextrem_v1.0.zip herunterladen  
2. ZIP in gewünschtes Verzeichnis extrahieren (Empfehlung: C:/Programme)  
3. livextrem.exe auswählen und Verknüpfung erstellen  
4. Verknüpfung auf Desktop legen  

Fertig.
